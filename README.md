# Full Stack Application: Flask API + Angular Frontend + MSSQL

Welcome to this full-stack application! This project is designed to demonstrate a complete application flow using a Python Flask backend, an Angular frontend, and a Microsoft SQL Server database.

## üìÇ Project Structure

This repository is divided into three main parts:

1.  **[Backend (PythonFlaskAPIBackend)](./PythonFlaskAPIBackend/README.md)**: The server-side API built with Python and Flask. It handles data processing and communicates with the database.
2.  **[Frontend (AngularFrontend)](./AngularFrontend/README.md)**: The client-side user interface built with Angular. It allows users to interact with the application.
3.  **[Database](./Database_README.md)**: The SQL scripts required to set up the Microsoft SQL Server database.

## üöÄ Getting Started

To get the application running, you will need to set up each component in the following order:

1.  **Database**: Set up the SQL Server database using the provided script. See [Database Setup Guide](./Database_README.md).
2.  **Backend**: Install dependencies and start the Flask API. See [Backend Setup Guide](./PythonFlaskAPIBackend/README.md).
3.  **Frontend**: Install dependencies and start the Angular application. See [Frontend Setup Guide](./AngularFrontend/README.md).
